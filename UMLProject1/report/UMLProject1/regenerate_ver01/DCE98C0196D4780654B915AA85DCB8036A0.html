public class Course

@Author: Beenish Jamil, Emily Vorek @Date: @Class: CS490 @Project: Course Scheduler @Class Description: This class constructs a Course object that contains GMU catalog information about a specific course. These course objects are used as the most basic building blocks for constructing a course schedule for n semesters for a specified degree program. Each course object is independent of an explicit degree program.
