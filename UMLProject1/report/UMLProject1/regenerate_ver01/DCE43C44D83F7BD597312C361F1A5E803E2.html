public class Degree

@Author: Beenish Jamil, Emily Vorek @Date: @Class: CS490 @Project: Course Scheduler @Class Description: This class creates Degree objects that store information about the requirements of degrees. It requires information on the types of courses required by the degree as well as the number of courses needed for each of those requirements. It also requires a catalog of all courses offered by the institution that are relavent to this degree as a parameter of its constructor. A degree object can take in a schedule object as its parameter and identify whether that schedule fulfills this degree's requirements or not.

public Course[0..*] getRequiredCourses(Map<String, Course> catalog, String selected[0..*])
!!!!!!!!!!! very brittle implementation: NEEDS REVIEW / REDOING Returns an ArrayList of Course object that is a subset of the catalog such that each requirement type is met. Example: If requirements are All CORE, 4 SENIOR_ELECTIVE, ALL GENED courses, the returned ArrayList will contain the specified number of all of those courses from the catalog. It is assumed that if a number, m > 0 is specified for the number of courses of a specific course type, then the actual number of offered courses in the catalog of that type will be a value, f, such that f > m . All courses in f are picked randomly using Java's default random number generator. @return a set of required course objects
