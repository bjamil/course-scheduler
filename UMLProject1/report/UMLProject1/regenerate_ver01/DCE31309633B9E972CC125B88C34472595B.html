public class Semester

@Author: Beenish Jamil, Emily Vorek @Date: @Class: CS490 @Project: Course Scheduler @Class Description: This class constructs a semester object, which is essentially a container of a group of course objects. The number of course objects that can be in each semester is bounded by the attributes maxCredits (default = 18) and minCredits (default = 12), each of which can be changed by the user.
